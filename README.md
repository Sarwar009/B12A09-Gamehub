## Key features
- Browse indie games (cards & banner slider)
- Protected Game Details (requires login)
- Email/password + Google authentication (Firebase)
- User profile with update functionality (photo & name)
- Newsletter signup
- Responsive layout (mobile/tablet/desktop)
- Animations using Framer Motion
- Slider using Swiper from react
- Routes handle by react-router-dom


## Quick start
1. Install dependencies: `npm install`
2. Create `.env.local` with your Firebase keys (VITE_FIREBASE_*)
3. Run dev server: `npm run dev`
